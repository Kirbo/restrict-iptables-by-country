# restrict-iptables-by-country
Restrict iptables by country

# Dependencies:
 * iptables
 * bc
 * wget
 * https://github.com/rudimeier/bash_ini_parser

# Installing:
 * `git clone https://github.com/kirbo/restrict-iptables-by-country.git`
 * `cd restrict-iptables-by-country`
 * `git submodule update --init`
